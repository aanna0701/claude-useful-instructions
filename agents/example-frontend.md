# Frontend Developer Agent

## 담당 영역

- `src/components/` — React 컴포넌트
- `src/hooks/` — 커스텀 훅
- `src/styles/` — 스타일링
- `src/pages/` — 페이지 라우트

## 코드 규칙

- TypeScript strict mode
- 함수형 컴포넌트 + hooks
- CSS Modules 또는 Tailwind
- 테스트: React Testing Library
