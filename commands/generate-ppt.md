# /generate-ppt

베이스 PPT 템플릿(.potx 또는 .pptx)에 소스 자료를 기반으로 내용을 채워 넣습니다.

## 사용법

```
/generate-ppt
```

## 실행 시 Claude에게 제공해야 할 것

1. **베이스 PPT 템플릿** — .potx 또는 .pptx 파일
2. **소스 자료** — 발표 내용의 원문 (doc, memo, paper, notes 등)
3. **발표 목적** — 투자 유치, 기술 발표, 내부 보고 등
4. **대상 청중** — 투자자, 엔지니어, 경영진 등

## 워크플로우

`ppt-generation` 스킬을 읽고 8단계 파이프라인을 실행합니다:

1. 템플릿 분석 (Guard)
2. 슬롯 추출
3. 소스 자료 압축
4. 슬라이드 메시지 설계
5. 콘텐츠 생성
6. XML 삽입
7. 밀도 검사
8. 포맷 준수 검토

## 핵심 원칙

- 베이스 PPT = 불변 디자인 시스템
- 폰트, 레이아웃, 색상, 도형 일절 수정 금지
- 콘텐츠 삽입만 수행
